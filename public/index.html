Query Processing Details

1. Time Converter

Datetime strings are translated in backend into **Unix Epoch Milliseconds** values required by Grafana.

2. Payload Builder

Request constructs a Grafana POST JSON payload with targets querying datasource UID.

3. Array Parser

Backend sanitizes multi-level arrays returned from Grafana metrics into clean coordinates [timestamp, value].

**Hephaestus Devops Portal v2** is a dynamic orchestration portal that allows direct Grafana HTTP API mapping.
